Session 3 — Parallel Computing with MPI

What You'll Learn

Tutorial 1 — MPI Scaling

  • MPI concepts: Scatterv, Reduce, ranks
  • Setup: module load openmpi + uv venv
  • 4 Python demos: 1→4→8 CPUs (1 node), then 8 CPUs (2 nodes)
  • C++ hybrid MPI + OpenMP demos
  • Key lesson: network latency kills multi-node performance

Tutorial 2 — Profiling

  • Live monitoring with sstat and htop
  • Post-job analysis: sacct, seff
  • Wall-time efficiency and memory efficiency

Tutorial 3 — Efficient High-throughput

  • Combined job (28% memory efficiency) vs. split jobs (75% + 70%)
  • SLURM job dependencies with afterok
  • Automatic dependency submission scripts

Prerequisites

  • Sessions 1 & 2 complete
  • uv available on the cluster
  • C compiler for Tutorial 1 C++ demo (mpic++)
Cluster context: ilifu uses Gigabit Ethernet between nodes, not InfiniBand. The multi-node scaling results in Tutorial 1 reflect this real-world constraint — a deliberate teaching point.
Next: Session 4 → introduces workflow orchestration with Nextflow and Cromwell for complex multi-step pipelines.
