Session 2 — Python Environments & Interactive Jobs

What You'll Learn

Tutorial 1 — Virtual Environments

  • Choose a Python version with module load
  • Create and activate a virtualenv
  • Install packages with pip in isolation
  • Register the environment as a Jupyter kernel

Tutorial 2 — Interactive Jobs

  • Difference between sinteractive and srun --pty bash
  • Running a 3-script demo workflow interactively
  • Passing parameters at runtime — impossible with static sbatch

Tutorial 3 — Job Arrays

  • Submit 20 tasks with a single sbatch command
  • Limit concurrency to 5 simultaneous jobs
  • Monitor progress with watch squeue
  • Understand structured log file naming

Prerequisites

  • Session 1 complete (or equivalent SLURM basics)
  • SSH access to the cluster
  • Repository cloned and cd session2/
Key skill: After this session you'll be able to manage Python environments and submit both interactive and batch array jobs — the foundation for all subsequent tutorials.
