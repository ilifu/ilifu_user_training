Session 1 — Introduction to SLURM

What You'll Learn

Tutorial 1 — Jupyter on ilifu

  • Launch JupyterHub from the ilifu portal
  • Navigate the notebook interface
  • Run cells, inspect outputs
  • Understand how Jupyter interacts with the cluster

Tutorial 2 — sbatch Basics

  • Anatomy of an #SBATCH script
  • Minimal vs. maximal job submission patterns
  • Monitoring jobs with squeue and sinfo
  • Singularity containers as an alternative environment

Prerequisites

  • SSH access to slurm.ilifu.ac.za
  • Basic Linux command-line (ls, cd, cat)
  • Repository cloned: git clone <repo>
No prior HPC experience required. Session 1 starts from scratch — bring your curiosity!
